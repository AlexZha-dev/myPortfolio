This project is designed as a personal portfolio website with a multilingual content structure and a clean GitHub Pages publishing flow.

It is not only a showcase of work, but also a case study that reflects the way I approach frontend development:

- content is separated from the UI layer;
- project texts are managed through Markdown;
- localization is treated as part of the architecture rather than an afterthought;
- the visual language is built around a calm dark-green palette.

### Why this case matters

This project reflects the direction I want to keep pushing: interfaces with character but without noise, systems that can grow without friction, and presentation where structure feels just as deliberate as the visuals.
